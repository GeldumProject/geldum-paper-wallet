# GELDUM PAPER WALLET

Basic Online / Offline Paper Wallet Generator

# Online

  - Upload to your server
  - Rename as index.html
  - Visit http://yourip

# Offline

  - Download to your computer
  - Disconnect from internet
  - Open file
  - Create wallet
